The Rosewood Gaming Engine

Ver 0.0.01a

What is this madness?

The Basics

The Rosewood Engine is a simple engine/framework for creating 2D, sprite-based games in javascript. Due to the limitations of current javascript engines, Rosewood is most suited for creating games in a style similar to those of the early 1990s (eg. Super Nintendo and Sega Genesis). Games created with Rosewood can be played in any moderately standards-compliant web browser ( Firefox, Chrome, Opera & Safari should do fine, maybe IE7+, but probably not IE6).

The Details

At its core, Rosewood is a Javascript Library, wrapped in an Object called 'rw'. The rw Object contains built-in functions for creating game menus, an animated runtime environment with collision detection and in-game overlays. The rw API allows for quick keybinding and animation controls and can be extended with new game logic. As of right now, Rosewood only has built-in support for top-down style gaming, but support for platform games is coming.

The Future

Server-side storage and AJAX calls are planned for the future as well as multiplayer support. Currently, Rosewood does not support canvas, but this may change.
